# نظام الفوترة — مؤسسة ليلى حكمي للمقاولات العامة

نظام فوترة يعمل **محلياً على جهازك** بدون تثبيت ولا خادم. افتح الملف `index.html` بالمتصفح (Chrome أو Edge موصى به).

## السجل التجاري

**4030384492**

## المميزات

- **فواتير برقم تسلسلي** تلقائياً (صيغة: `INV-السنة-رقم`).
- **فاتورة ضريبية أو عادية** — عند تفعيل الضريبة يُحسب **ضريبة القيمة المضافة** على المجموع قبل الضريبة حسب النسبة في الإعدادات (افتراضياً **15%**).
- بنود ديناميكية: **البيان، الكمية، سعر الوحدة، الإجمالي**.
- **معاينة احترافية** جاهزة للطباعة بحجم **A4**.
- **تصدير PDF** للفاتورة.
- **سجل الفواتير** مع بحث وتصفية.
- **إدارة العملاء** وربطهم بالفواتير.
- **لوحة تحكم** و**تقارير** مع رسوم بيانية.
- **نسخ احتياطي / استعادة** بصيغة JSON من الإعدادات.

## طريقة التشغيل

1. انسخ مجلد `LaylaHekmiInvoices` إلى أي مكان (مثلاً سطح المكتب).
2. انقر نقراً مزدوجاً على **`index.html`** لفتحه في المتصفح.

> **ملاحظة:** البيانات تُحفظ في **LocalStorage** الخاص بالمتصفح. إذا مسحت بيانات الموقع ستُفقد الفواتير ما لم تصدّر نسخة احتياطية.

## اختصارات

- داخل نموذج الفاتورة: **Ctrl+S** — حفظ.
- داخل معاينة الفاتورة: **Ctrl+P** — طباعة.

## تغيير الشعار وبيانات المؤسسة

من **الإعدادات**: يمكنك رفع شعار، تعديل الاسم، العنوان، الرقم الضريبي للمؤسسة، ونسبة الضريبة، ونص الشروط والتذييل.

## المكتبات (محلية داخل `lib/`)

- jsPDF + html2canvas — تصدير PDF.
- Chart.js — الرسوم في لوحة التحكم والتقارير.

## الدعم الفني

لأي تعديل على التصميم أو الحقول الإضافية، يمكن تطوير المشروع لاحقاً بناءً على احتياجات المؤسسة.
